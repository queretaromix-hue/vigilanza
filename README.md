# Vigilanza Elite Private Security

This is the official landing page for **Vigilanza**, a premier private security service provider specializing in gated communities and professional security personnel.

## 🛡️ Company Mission
Vigilanza is dedicated to providing uncompromising safety and peace of mind for luxury residential areas and corporate facilities through a combination of elite guard forces and cutting-edge security technology.

## 🚀 Technical Stack
- **Vite**: Ultra-fast build tool and dev server.
- **Vanilla JavaScript**: Lightweight and high-performance.
- **Premium CSS**: Custom design system with glassmorphism and modern animations.

## Getting Started

1. **Navigate to the project directory**:
   ```bash
   cd /Users/diegopenasanchez/.gemini/antigravity/scratch/vigilanza
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
